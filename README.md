## Getting Started

How to test ?

```bash
git clone https://github.com/Thomas-Sohier/jellyflut.git
cd jellyflut
flutter packages get
flutter packages upgrade
flutter run
```

How to build ? (Android)

```bash
flutter build apk --release
```
